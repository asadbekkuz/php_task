CONFIGURATION
-------------

### Database

Edit the file `connection.php` with real data, for example:

```php
$host = 'your_server';
$port = 'your_server_port';
$username = 'your_db_username';
$password = 'your_db_password';
```

* composer update for load dependencies
~~~
composer update
~~~
